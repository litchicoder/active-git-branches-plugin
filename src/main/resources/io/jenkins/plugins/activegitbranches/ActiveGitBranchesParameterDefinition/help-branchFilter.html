A regular expression pattern to filter branch names.

Only branches that match the pattern will be included in the list.

Examples:

  • .* - Match all branches (default)
  • feature/.* - Only feature branches
  • release/.* - Only release branches
  • (main|master|develop) - Only main, master, or develop
  • (?!release/).* - All branches except release branches

Leave empty to include all branches.
